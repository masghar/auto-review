1 Prerequisites

  • Browser: Google Chrome, Microsoft Edge, Brave, Arc, or any Chromium-based browser.
  • Google account with access to Google Business Profile (at least one verified business location).
  • Gemini API key — free to obtain; instructions in the next section.
  • Developer Mode in Chrome — only needed if installing from source (not required for Chrome Web Store installs).

2 Getting Your Free Gemini API Key

Step-by-Step

  1. Open https://aistudio.google.com/app/apikey — sign in with any Google account.
  2. Click "Create API key".
  3. Select an existing Google Cloud project or click "Create new project".
  4. Copy the generated key — it starts with AIza and is approximately 39 characters long.
  5. Store it somewhere safe (like a password manager). Treat it like a password — anyone with your key can use your quota.
You do not need a credit card to get a free Gemini API key. The free tier is activated automatically.

Free Tier Rate Limits (Gemini 2.5 Flash)

LimitAmountWhat it means for you
Requests per minute (RPM)15Up to 15 review replies per minute
Requests per day1,500Up to 1,500 review replies per day
Tokens per month1,000,000Each reply uses ~500 tokens — about 2,000 replies/month free

Practical advice: For most small and medium businesses (under 50 pending reviews), the free tier is more than sufficient. The extension will show an orange "Rate limited — wait a moment and retry" message if you hit the RPM limit during a large bulk run. Simply wait 60 seconds and click Reply All again — it picks up exactly where it left off.

Upgrading to a Paid Key

If your review volume regularly exceeds the free tier, you can upgrade at the same Google AI Studio page. Paid billing is per token — at roughly 500 tokens per reply, the cost is a fraction of a cent per response. For most businesses the free tier is permanent.

3 Installing the Extension

From Chrome Web Store (Recommended)

  1. Search for "GMB Review AI" on the Chrome Web Store, or open the direct listing link.
  2. Click "Add to Chrome".
  3. Confirm the permissions prompt — click "Add extension".
  4. The star icon (★) appears in your Chrome toolbar. Done.

From Source (Developer Install)

  1. Download or clone the repository to your computer.
  2. Open Chrome and go to chrome://extensions.
  3. Enable "Developer mode" using the toggle in the top-right corner.
  4. Click "Load unpacked" and select the folder containing manifest.json.
  5. The GMB Review AI star icon appears in the Chrome toolbar.
After updating the source files, go to chrome://extensions and click the refresh icon on the GMB Review AI card, then reload your Google Business Profile tab. The version number in the panel header confirms you have the latest build.

4 First-Time Setup

Step 4.1 — Open the Panel

  1. Navigate to https://business.google.com/reviews.
  2. The GMB Review AI side panel opens automatically on the right side of the page. If it doesn't, click the ★ FAB button (floating star button at the bottom-right of the page) or click the extension icon in the Chrome toolbar.

Step 4.2 — Add Your Gemini API Key

  1. Click the Settings tab in the panel.
  2. Paste your Gemini API key into the "Gemini API Key" field.
  3. Select your default Reply Tone (you can change this per session).
  4. Click "Save All Settings".

Step 4.3 — Add Your Business Profile(s)

  1. In the Settings tab, click "+ Add Profile".
  2. A new profile card appears — expand it and fill in all fields:

5 Generating & Posting Replies

Scan Reviews

  1. Click the Reviews tab in the panel.
  2. Click "Scan Reviews". The extension scans the page and loads all pending reviews into the panel list.
  3. Each card shows the reviewer name, star rating, a text snippet, the date, and a purple badge showing which business profile was matched.

Generate One Reply

  1. Select the Reply Tone from the dropdown at the top of the Reviews tab.
  2. Click "Generate" on any review card.
  3. Gemini writes a 1–2 sentence, personalised reply in about 2–3 seconds.
  4. Read the reply — it is directly editable in the card. Change anything you like.
  5. Click "Post". The extension scrolls to the review, clicks Google's Reply button, fills the textarea, and submits automatically.

Reply All (Bulk Mode)

  1. After scanning, click "Reply All".
  2. The extension automatically generates and posts a reply for every pending review on the current page.
  3. When it finishes the current page, it clicks Google's "Next page" button and continues — working through all pages without any input from you.
  4. Watch the Activity Log (expand it at the bottom of the panel) for real-time progress — every step is logged.
  5. Click "Stop" at any time to pause the process.
Rate limit during bulk runs: If you have many reviews and hit the Gemini free tier limit (15 requests/minute), an orange message will appear in the log. The extension pauses — simply wait 60 seconds, then click Reply All again to continue.

6 Managing Multiple Business Locations

If you manage more than one business, add a profile for each location in Settings. The extension automatically matches each review to the correct profile based on the business name shown on the Google review card.

The purple badge on each review card in the panel shows which profile was matched (e.g., "Serenity Spa" or "House of Salons"). Always check these badges after scanning to verify the matching is correct before running Reply All.

If a review doesn't match any profile (e.g., a new location not yet in Settings), it will fall back to your first profile. Add a profile for the new location and re-scan.

The extension reads whatever business name Google shows in the review card section headers. Open your Google reviews page, look at what text appears above or near each set of reviews, and use that as your Business Name field value.

9 Frequently Asked Questions

Does this cost anything?

The extension is completely free. You need a Gemini API key — the free tier provides 1,500 requests/day which covers most small and medium businesses. There is no subscription, no in-app purchase, and no premium tier.

Does the extension see my Google account password?

No. It works entirely through your existing browser session — the same session you already use to manage Google Business Profile. The extension never asks for, sees, or transmits your Google login credentials.

Is my data stored on your servers?

No. Your API key and all settings are stored only in Chrome's local storage (chrome.storage.local) on your own computer. Review text is sent directly from your browser to Google's Gemini API using your own key. The extension developer has zero access to any of this data.

Can I use this on multiple computers?

Settings are stored locally per browser profile. Chrome's chrome.storage.local is intentionally not synced across devices. You will need to re-enter your API key and profiles on each device. This is by design — it keeps your API key off the cloud.

What happens if Google changes their reviews page?

The extension reads Google's DOM structure to find review cards and reply buttons. If Google makes significant structural changes, some features may temporarily stop working. Check for an updated version of the extension in that case.

Can I edit the generated reply before posting?

Yes — the generated text appears in an editable text area inside the review card. Click anywhere in the text to edit it before clicking Post.

Does Reply All post immediately, or does it show me each reply first?

Reply All generates and posts each reply automatically without pausing for your review. If you want to check each reply before posting, use the one-at-a-time Generate → Post flow instead.

What Gemini model is used?

Gemini 2.5 Flash — Google's fastest model, optimised for short creative writing tasks. It generates review replies in 2–3 seconds each. Thinking mode is disabled to keep replies fast and under 50 words.
